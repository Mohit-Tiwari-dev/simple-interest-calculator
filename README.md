# Simple Interest Calculator

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

## Formula Used

Simple Interest = P × R × T / 100

## Input

- Principal amount
- Rate of interest
- Time period

## Output

- Simple Interest
